# AWS-Based ETL Pipeline Project

## Overview
This project demonstrates an end-to-end ETL pipeline using AWS services.

## AWS Services Used
- Amazon S3
- Amazon RDS (MySQL)
- AWS IAM

## Technologies
- Python
- Pandas
- Boto3
- SQLAlchemy

## Workflow
1. Extract data from S3
2. Transform using Pandas
3. Load cleaned data into RDS
